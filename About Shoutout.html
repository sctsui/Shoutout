About Shoutout

Description

Shoutout is a web application designed to create a more inclusive work environment,through peer-to-peer recognition,awareness building,and positive reflection.

Purpose

Shoutout was made to combat imposter syndrome in the workplace in order to foster a more accommodating environment and encourage diversity.
